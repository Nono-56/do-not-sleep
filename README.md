# Do Not Sleep

Windows PC がスリープに入らないよう、一定間隔でキー入力または微小なマウス移動を送る軽量アプリです。

## 使い方

1. 配布された zip ファイルを任意のフォルダに展開します。
2. `dist\do-not-sleep.exe` を起動します。
3. 動作方式を選びます。
   - キー入力: スクリーンキーボードから送信するキーを選択
   - マウス移動: カーソルをわずかに動かして元に戻す方式
4. 実行間隔を設定します。
   - 分指定: `1` から `240`
   - 秒指定: `1` から `3600`
5. 必要なら終了日時を `YYYY-MM-DD HH:MM` 形式で設定します。
6. `開始` を押します。停止するときは `停止` を押します。

## 主な機能

- キー入力 / マウス移動の切替
- GUI 上のスクリーンキーボードからキー選択
- `F13` から `F16` を含むキー送信
- 分または秒単位の実行間隔設定
- 次回実行までの秒数表示
- 終了日時による自動停止
- 設定の自動保存と次回起動時の復元

## 動作環境

- Windows 10 / 11

## 設定保存

設定は exe と同じ場所の `config\config.json` に保存されます。

- `config` ディレクトリまたは `config.json` が存在しない場合は自動生成されます。
- `config.json` が壊れている場合は、デフォルト設定で再生成されます。

## 注意事項

- 擬似入力は Windows API の制約やセキュリティ製品の影響を受けることがあります。
- 管理者権限で動作しているアプリに対しては、通常権限の本アプリから入力が届かない場合があります。
- `Win`、`Power`、`Sleep`、`Wake`、`PrintScreen` などは副作用回避のため対象外です。
- 終了日時は現在より未来の日時のみ指定できます。
- 動作中に設定を変更した場合、新しい間隔は次回入力実行後から反映されます。

## 開発者向け

Python から起動する場合:

```powershell
python -m app
```

ビルドする場合:

```powershell
python -m pip install -r .\requirements.txt
.\build.ps1
```

生成物は `dist\do-not-sleep.exe` です。
